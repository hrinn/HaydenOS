# HaydenOS
A kernel for x64. This is a project for Dr. John Bellardo's Implementing OS class at Cal Poly, SLO.

![Snakes!](https://i.imgur.com/E6uxOJ1.gif)

## Features
- Interrupts
- Virtual Memory
- Threads / Scheduling
- Virtual Filesystem
- User mode

## Device Support
- VGA Text Mode
- Keyboard
- UART
- ATA Drives
