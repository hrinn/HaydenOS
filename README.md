# HaydenOS
A hobby kernel for x64. This project was started in Dr. John Bellardo's Implementing OS class at Cal Poly, SLO.

![Snakes!](https://i.imgur.com/E6uxOJ1.gif)

## Features
- Interrupts
- Virtual Memory
- Threads / Scheduling
- Virtual Filesystem
- User Mode

## Device Support
- VGA Text Mode
- Keyboard
- UART
- ATA Drives
